MY READ

Both now live in the same family.

Plates and the Supervised cube share the 30° isometric axis, the single‑ink palette, and the opacity ladder. You can ship either and the system stays coherent — tables, badges, and confidence signals don't need to change.

Supervised cube carries more product meaning (the thesis of "one unit surfaced") but asks more of the viewer. Plates is simpler, less editorial, easier at 16px.

Pick one — landing page and full system sheet re‑skin to match.

CHANGES FROM v1

Violet is gone from the mark.

The accent color (violet 300h) is still reserved in the system — for agent actions, confidence signals, and the autonomy dial — but it no longer appears in the logo. The mark is a two‑tint architectural glyph; the agent layer adds the color when it earns it.

Inset opacity. Dark‑ink inset at 100% on light bg; on dark bg it inverts to white at ~35% so it still reads as "punched out" without introducing a new hue.

Same grid, same angle, same wordmark. Only the palette narrowed.
