Principle
Off‑nominal is red. Within‑range is neutral.
NASA Open MCT. Color manages attention toward exceptions — it doesn't decorate the normal state. Green is used sparingly, only when positive confirmation is the message (completed, approved).
Principle
Color is never the only channel.
Every status combines color + icon + text label. Red‑green CVD is the most prevalent form of color blindness; every badge must survive grayscale.
Principle
Violet is reserved. It means agent.
No other UI element uses the 300° hue. Wherever you see violet — avatars, badges, audit rows — an agent was the actor.

05 · AGENT STATES · 10

The state machine, made visible.

Every agent action lives in one of ten states. Each has a distinct badge, a required action, and a specification of the minimum data a reviewer needs to decide within 10–30 seconds.
Light
Suggested by Agent

Pending Approval · 14m
Countdown active. Human must decide before SLA expires. Show: assignee, time remaining before escalation. Actions: approve / reject / delegate.
Approved
Human signed off. Agent is cleared to execute. Show: approver, timestamp. No action required.
Executing · 43%
Tools are being called. Show: current step, % complete, ETA, budget consumed. Actions available: pause / stop.
Completed
Action finished, receipt available. Show: what changed, diffs, rollback link. Optional: review receipt.

12 · PRINCIPLES

The 10 rules the whole system follows.

If a screen seems to violate one of these, the screen is wrong. Not the principle.
01 · Supervise
Humans supervise. Agents operate.
The default UI is optimized for oversight. Execution happens in the background, surfacing only what requires judgment.
02 · Propose
Every agent proposes before it commits.
Intent and execution are separated. AUTO, LOG, or REQUIRE_APPROVAL — one of the three, always.
03 · Exception‑first
The default state is invisible.
Routine operations are silent. Attention is managed toward the off‑nominal.
04 · Earn autonomy
L1 → L5 is a ladder with data checkpoints.
Promotion requires >95% accuracy over hundreds of reviewed interactions.
05 · Calibrate trust
Confidence + rationale + evidence + undo.
Every consequential proposal carries all four. Never three.
06 · Audit
Every agent + human event is first‑class.
Audit log is a surface, not a debug tool. Meets EU AI Act Art. 12(1).
07 · Task‑centric
Screens are organized by what needs attention.
Not by database tables. The Units table exists, but it isn't the homepage.
08 · Least privilege
Each agent has a scoped, revocable identity.
Delegated from the human supervisor. No service accounts.
09 · Plan‑level review
Approve the plan, not each step.
Plan‑based strategies cut problematic actions 75%. Step rubber‑stamping exhausts reviewers.
10 · Dual channel
Every surface serves humans and agents.
Machine‑readable intents and receipts sit alongside the human UI. MCP‑native.
